# Employee Payroll System

The Employee Payroll System is a C program that calculates the salaries of employees based on their basic pay, dearness allowance (DA), and house rent allowance (HRA).

## Introduction

The Employee Payroll System allows you to input the following details for multiple employees:
- Employee name (up to 9 characters)
- Age
- Basic pay

It then calculates the DA, HRA, and total salary based on predefined percentages and displays the results.

## Input Format

The Employee Payroll System expects input in the following format:

1. Start by entering the number of employees for whom you want to calculate salaries.

2. For each employee, provide the following details:
   - Employee name (up to 9 characters)
   - Age
   - Basic pay

   Ensure that you enter accurate and valid information for each employee.
## Output Format

The Employee Payroll System displays the calculated details for each employee in the following format:

- Name
- Age
- Basic Pay
- Dearness Allowance (DA)
- House Rent Allowance (HRA)
- Total Salary
