# Pizza Order System

The Pizza Order System is a Python program that allows customers to choose from a selection of pizza sizes and add extra cheese to their order. The program calculates the total cost based on the customer's choices and provides the final price.

## Introduction

The Pizza Order System program simplifies the process of ordering pizza by providing a menu of pizza sizes and the option to add extra cheese. It calculates the total cost and displays the final price to the customer.
